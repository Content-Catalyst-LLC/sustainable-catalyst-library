# Release Notes — Sustainable Catalyst Library v4.0.4

## Fixed

- Removed the undefined `META_SOURCE_ATTACHMENT` class constant reference.
- Updated source-record serialization to use `META_PDF_ID`.
- Added a compatibility alias for older extension code.
- Prevented the Research Library from terminating in the Public API / Export /
  Federation source-record path.
- Preserved all existing Knowledge Library, Foundations, PDF, OCR, API, and
  institutional records.

## Upgrade

Upload the generated WordPress plugin ZIP and choose **Replace current with
uploaded**. No separate import or migration step is required.
