# Sustainable Catalyst Foundations v2.1.5

## Live Catalog Family and Type Synchronization

The records were published, but ten remained Unassigned in the current public
Document Repository. The Foundations page filters the public family taxonomy,
so those records were excluded.

v2.1.5 assigns every First Edition record to:

- Family: Foundations
- Document Type: Foundation Document
- Lifecycle: Current

It also preserves the original Knowledge Library Foundations collection and
Institutional Foundations category. Provisioning is not considered complete
until all 13 records are publicly catalog-ready.
