# Library v1.13.2 Index Scanner Route Repair

Upload and activate `sustainable-catalyst-library-v1.13.2.zip`, replacing v1.13.1.

Then open:

`/wp-admin/admin.php?page=sc-library-scanner`

The page should now load under **SC Library → Index Scanner**.

Recommended first scan:

1. Select all configured post types.
2. Choose **Complete safe rebuild**.
3. Use a batch size of **50**.
4. Start the scan.
5. Clear WordPress, page-builder, Cloudflare, and browser caches after completion.

The scanner remains independent of Render, PostgreSQL, workspace synchronization, and document production.
